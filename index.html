Vad är selektorer?

Selektorer används i CSS för att välja vilka element man vill ändra och manipulera i sitt HTML-dokument.

Basala Selektorer

Dessa selektorer är de vanligaste och används oftast.

element

Väljer alla element som har det angivna element-namnet

p { background-color: yellow; }
Resultat

.class

Väljer alla element som har det angivna klass-namnet.

.intro { background-color: yellow; }
Resultat

#id

Väljer det element som har det angivna id-namnet.

#firstname { background-color: yellow; }
Resultat

*

Väljer alla element.

* { background-color: yellow; }
Resultat

Kombinatörer

Kombinatörer används för att kombinera selektorer med andra selektorer, för att t.ex. kunna ha fler selektorer som följer samma kodblock.

, - element,element

Används för att ange fler selektorer.

h1, p { background-color: yellow; }
Resultat

(mellanslag) - element1 element2

Väljer alla element2 inuti ett annat element1.

div p { background-color: yellow; }
Resultat

> - element1 > element2

Väljer alla element2 som är barn till ett annat element1.

div > p { background-color: yellow; }
Resultat

+ - element1 + element2

Väljer alla element2 som är placerade direkt efter ett annat element2.

div + p { background-color: yellow; }
Resultat

~ - element1 ~ element2

Väljer alla element2 som kommer efter element1.

div + p { background-color: yellow; }
Resultat

Pseudo Klasser

Dessa används när man ska välja någonting som inte har den aktuella informationen i HTML-koden, när någonting som är i ett speciellt tillstånd ska väljas.

:link

Väljer en länk som inte har klickats på.

a:link { background-color: yellow; }
Resultat

:visited

Väljer en länk som har besökts.

a:visited { color: pink; }
Resultat

:active

Väljer en länk som är aktiv(när man klickar på den).

a:active { background-color: yellow; }
Resultat

:hover

Väljer en länk som muspekaren svävar över.

a:hover { background-color: yellow; }
Resultat

:focus

Väljer någonting som har fokus på sig, detta kan vara ett input fält t.ex.

input:focus { background-color: yellow; }
Resultat

:target

Väljer någonting som har blivit länkat till, t.ex. en paragraf som länkats till via en länk.

:target { border: 2px solid #D4D4D4; background-color: #e5eecc; }
Resultat

:enabled

Väljer alla element som är aktiverade. används oftast i forms.

input[type="text"]:enabled { background: #ffff00; }
Resultat

:disabled

Väljer alla element som inte är aktiverade. används också oftast i forms.

input[type=text]:disabled { background: #dddddd; }
Resultat

:checked

Ändrar storlek på alla input-element som är incheckade.

input:checked { height: 50px; width: 50px; }
Resultat

:lang()

Ändrar färg på varje p element med attribut "it"(italienska)

p:lang(it) { background: yellow; }
Resultat

:indeterminate

Väljer element som är i ett tillstånd som är obestämd.

input:indeterminate { box-shadow: 0 0 1px 1px red; }
Resultat

Pseudo element

Dessa används för att style speciella delar av ett element. t.ex. första bokstaven eller raden i ett element.

::before

Infogar "Read this: " innan det angivna elementet.

p::before { content: "Read this: "; }
Resultat

::after

Infogar "- Remember this" efter det angivna elementet.

p::after { content: " - Remember this"; }
Resultat

::first-line

Ändrar färg på första raden i p elementet.

p::first-line { background-color: yellow; }
Resultat

::first-letter

Ändrar storlek och färg på första bokstaven i p elementet

p::first-letter { font-size: 200%; color: #8A2BE2; }
Resultat

Barn-selektorer

Barn selektorer används nästan uteslutande för att välja och arbeta med element som är barn till en förvald selektor, därav namnet.

:first-child

:first-child selektorn används för att välja ett element, enbart om det är det första barnet till ett angivet element.

p:first-child{background-color: yellow;}
Resultat

:last-child

:last-child selektorn används för att välja ett element, enbart om det är det sista barnet till ett angivet element.

p:last-child {background: #ff0000;}
Resultat

:first-of-type

:first-of-type selektorn används för att välja det första barn-elementet av sin typ.

p:first-of-type {background: red;}
Resultat

:last-of-type

:last-of-type selektorn används för att välja det sista barn-elementet av sin typ.

p:last-of-type {background: red;}
Resultat

:nth-child(n)

:nth-child(n) selektorn väljer det element som har n(number) plats till dess förälder.

p:nth-child(2) {background: red;}
Resultat

:nth-last-child(n)

:nth-last-child(n) selektorn väljer det element som har n(number) plats till dess förälder, fast i omvänd ordning.

p:nth-last-child(2) {background: red;}
Resultat

:nth-of-type(n)

:nth-of-type(n) selektorn väljer det element som har plats (n) av en speciell typ, till en förälder.

p:nth-of-type(2) {background: red;}
Resultat

:nth-last-of-type(n)

:nth-last-of-type(n) selektorn väljer det element som har plats (n) av en speciell typ, till en förälder. Räknat från det sista barnet.

p:nth-last-of-type(2) {background: red;}
Resultat

:only-child

:only-child selektorn väljer varje element som är det enda barnet till dess förälder.

p:only-child {background: red;}
Resultat

:only-of-type

:only-of-type selektorn väljer varje element som är det enda barnet av sin typ till en förälder.

p:only-of-type {background: red;}
Resultat

Attribut-selektorer

Attribut-selektorer används uteslutande för att arbeta med element genom deras attribut.

[attribute]

[attribute] selektorn används för att välja ett element med en speciell attribut.

a[target] {background-color: yellow;}
Resultat

[attribute=value]

[attribute=value] selektorn används för att välja ett element med ett speciellt attribut och värde.

a[target=_blank] {background-color: yellow;}
Resultat

[attribute~=value]

[attribute~=value] selektorn används för att välja ett attribut-värde som innehåller ett speciellt ord.

[title~=flower] { background-color: yellow;}
Resultat

[attribute|=value]

[attribute|=value] selektorn används för att välja ett attribut-värde som börjar med ett speciellt värde.

[lang|=en] {background-color: yellow;}
Resultat

[attribute$=value]

[attribute$=value] selektorn väljer alla element som har ett attribut värde som slutar med det angivna värdet.

div[class$="test"] {background: #ffff00;}
Resultat

[attribute*=value]

[attribute*=value] selektorn väljer alla element som innehåller det angivna attribut-värdet.

div[class*="test"] {background: #ffff00;}
Resultat

[attribute^=value]

[attribute^=value] selektorn används för att välja ett attribut-värde som börjar med ett värde men får efterföljas av andra ord direkt. t.ex går det bra att ha div[class^="test"] och sedan ha en div med en class som heter test2. Detta går inte med [attribute|=value]-selektorn.

div[class^="test"] {background: #ffff00;}
Resultat

Input-selektorer

Input-selektorer används nästan uteslutande för att manipulera element i former, främst input elementet.

:optional

:optional selektorn används för att välja element som inte har attributen required.

input:optional {background-color: yellow;}
Resultat

:out-of-range

:out-of-range selektorn väljer dem input element med ett värde utanför specificerad räckvidd(range).

input:out-of-range {border: 2px solid red;}
Resultat

:in-range

:in-range selektorn väljer dem input element med ett värde innanför specificerad räckvidd(range).

input:in-range {border: 2px solid yellow;}
Resultat

:default

:default selektorn väljer det inputfält som är förutbestämt(checked).

input:default {box-shadow: 0 0 1px 1px red;}
Resultat

:read-only

:read-only selektorn väljer dem input element med “readonly” attributen.

input:-moz-read-only {background-color: yellow;}
input:read-only {background-color: yellow;}
Resultat

:read-write

:read-write selektorn väljer dem input element som INTE har “readonly” attributen.

input:-moz-read-write {background-color: yellow;}
input:read-write {background-color: yellow;}
Resultat

:required

:required selektorn väljer dem input element med attributen required.

input:required {background-color: yellow;}
Resultat

:valid

:valid selektorn väljer alla input element med ett giltigt värde.

input:valid {background-color: yellow;}
Resultat

:invalid

:invalid selektorn väljer alla input element med ett giltigt värde.

input:invalid {border: 2px solid red;}
Resultat

::placeholder

::placeholder selektorn väljer dem input element med placeholder text. Placeholder selektorn behandlas annorlunda beroende på webbläsaren som används. Därför måste man vara vaksam över hur man använder denna selektor.

::-webkit-input-placeholder {color: red;}
::placeholder {color: red;}
Resultat

Diverse-selektorer

Dessa diverse selektorer är selektorer som inte riktigt passade in i någon av de andra grupperna.

:empty

:empty selektorn väljer element som är tomma, element som inte har några barn och inget innehåll.

p:empty {width: 100px;height: 20px;background: #ff0000;}
Resultat

:root

:root selektorn väljer root-elementet av ett dokument. t.ex. html elementet i ett HTML-dokument.

:root {background: #ff0000;}
Resultat

:not(selector/element)

:not(element/selector) selektorn väljer alla element som inte är av det angivna elementet/selektorn.

:not(p) {color: #ff0000;}
Resultat

:selection

::selection selektorn väljer den delen av ett element som har blivit markerat av användaren.

::-moz-selection {color: red;background: yellow;}
::selection {color: red;background: yellow;}
Resultat
